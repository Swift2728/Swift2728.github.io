OSX iTerm2 常用快捷键

常用快捷键

光标移动行首/尾

Ctrl + A / E

清屏操作

Command + K; Ctrl + L

删除光标前/后的内容

Ctrl + W / K

新建标签

Command + T

屏幕垂直分割，水平分割 (相对于分割线而言)

Command + D， Command + Shift + D

关闭当前页

Command + W # 类似 exit

左右切换tab

Command + <- / ->

历史命令

Command + Shift + H

给当前tab设置标题

zsh
echo -ne “\e]1;title\a”
